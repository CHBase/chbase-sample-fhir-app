    eq2013-01-14
    lt2013-01-14T10:00
    gt2013-01-14T10:00
    ge2013-03-14
    le2013-03-14
    sa2013-03-14
    eb2013-03-14
    ap2013-03-14 and 2013-03-14
https://www.hl7.org/fhir/search.html#date

Code can be either Snomed or Loinc codes .
It can be represent just with codes or by giving the url reference of the code format 
For snomed codes, 
i.e"code=http://snomed.info/sct|434912009" for blood glucose or simply "code=434912009".
For loinic codes, 
"code=http://loinc.org|3137-7" for height or simply "code=3137-7".
